# SiteHealth

Website health scanner with compare, watch, and fix mode.

## One command
`ash
pip install sitehealth
sitehealth example.com --fix
sitehealth --compare site1.com site2.com
sitehealth --url https://x.com --watch 3600
`

## What it checks
- HTTPS cert expiry  - robots.txt  - sitemap.xml  - Title/Desc/JSON-LD  - Load speed

## Install
`ash
pip install sitehealth
`
